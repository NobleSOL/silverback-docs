# Getting Started

Everything you need to start using Silverback DEX.

---

## Prerequisites

Before you begin, make sure you have:

**For Base Network:**
- A Web3 wallet (MetaMask, WalletConnect, Coinbase Wallet, etc.)
- ETH on Base network for gas fees
- Tokens you want to trade

**For Keeta Network:**
- Keythings wallet browser extension
- KTA tokens for transaction fees
